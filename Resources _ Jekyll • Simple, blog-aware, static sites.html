Shells

The Bridge Between Users and Linux (There are a lot of shells, but in this page we focus on bash),To learn this ,you have to know the commands on Linux terminal

Shell ScriptPermalink

  1. First Line(Shebang): In this line we use #！ and add the path of interpreter after it.

    Such as 

    #! /bin/bash
  2. Type your code to the following line 
    #! /bin/bash 
echo "Hello World!"
  3. To launch the bash script
    1. launch as a *.sh(On the terminal). 
      chmod +x *.sh
./*.sh
    2. As a parameter 
      /bin/sh test.sh
/bin/php test.php
    3. Recommend "菜鸟教程",to learn more Linux terminal command.

A real LinuxPermalink

  1. Get a USB for system

    2. Empty and pack up the data before format your usb for Linux system.
    Download the mirror of the Release version
    Use Ultra ISO to open and burn it on your USB

  2. Create a disk partition for the new system(which will disappear on your windows file manage tool)

    3. Please find help on the Internet

  3. Set your Computer to open by usb driver

    4. Make sure your usb is Plugged in your laptop
    Restart your computer and open the BIOS
    Set your USB driver to be opened at first
    Restart

  4. Distribute the disk space(Make sure you choose the zone you set up in step 2)

    5. Follow the instruction and allocate the space by yourself.
    Personally, there are only '/' '/home' '/boot' 'swap',four contents need to be initialized
